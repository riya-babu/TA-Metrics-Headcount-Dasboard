# Talent Acquisition Metrics & Headcount Dashboard

##  Project Overview
This project features a fully functional, recruiter-ready Excel workbook designed to streamline talent acquisition operations, monitor candidate pipelines, and provide real-time updates on active organizational vacancies.

##  Features Implemented
* **Dynamic Applicant Tracking:** Structured an initial database monitoring 16 mock candidates across technical panels, system design rounds, and offer stages.
* **Automated Data Validation & Workflows:** Built conditional formatting rules for recruitment coordinators, highlighting urgent actions (`Action Required`) and upcoming follow-ups (`Follow-up Due`).
* **Summary Metrics Dashboard:** Utilized mathematical functions (`SUM` and cell logic) to cleanly track target headcounts, accepted offers, and remaining openings without circular references.
* **Candidate Volume Analysis:** Developed structured Pivot Tables and dynamic Pivot Charts to visually map talent volume across critical engineering and HR roles.
